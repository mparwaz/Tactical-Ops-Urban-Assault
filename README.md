## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

Core Pillars
Procedural Combat Zones: No two matches are the same. The engine generates massive 10,000x10,000 unit maps for BR or tight, competitive lanes for traditional modes, ensuring fresh tactical challenges every round.
Deep Arsenal & Operator System: Choose from 10 unique Operators like the Juggernaut or Ghost, each with specific stat multipliers. Master 11 distinct weapons, ranging from the high-accuracy Intervention Sniper to the room-clearing AA-12 Automatic Shotgun.

Adaptive 2025 Technology:
True Portrait Mode: Optimized for mobile play with a vertical Field of View and specialized touch controls.
Screen Orientation Lock: Uses the latest screen.orientation.lock() API to force an immersive portrait experience on supported mobile devices.
PWA Ready: Can be installed directly from the browser to your home screen for an app-like experience.
